Animations with Angular

Exit & Entrance Animations

Entrances and exits come in different flavors. You can combine the two in various different combinations to achieve different effects. The primary thing to make note of is that you can not layer two different entrance or exit animations on the same element. The one that is written last will take priority.

Some classes have been provided for you as syntactic sugar, such as pb-fade. We've provided this class for convenience, but it could also be written as pb-fadeIn and pb-fadeOut.

Combine Entrance/Exit animations with Emphasis animations

While you can not layer multiple entrance or exit animations, you can layer on emphasis animations. By default adding an emphasis animation along with an exit and entrance animation will cause the emphasis animation to fire once the element's entrance animation has finished.

Displacement Issues - Cross Fading
