What is Angular Ui Bootstrap?

The default controls supplied with Boostrap are simple to use, require no additional Javascript development and are configured and used completely from the HTML markup.

For Angular JS development, there is a set of native AngularJS directives based on Bootstrap's markup and CSS which are styled to match the Design System. For more technical information, installation and usage instructions, please see the Angular UI Bootstrap site.
