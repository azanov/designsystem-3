Design System

Overview

The following AngularJS directives make implementing common controls easier:

  • Checkboxes
  • Radio Buttons

Once you've downloaded the Design System to your application, add the components script to your page. The following example assumes that the Design System was installed using Bower.

To use the Design System components, you need to add the component module as a dependency in your application as follows.

angular.module('app', [ 'pb.ds.components' ]);

Checkboxes

The checkbox directive provides an easier implementation of the Design System styled checkbox controls. The directive uses a element tag <pbds-checkbox></pbds-checkbox> with the following attributes.

Attribute Type Default Description
label string none The text label for the checkbox input. When using localized strings with angular-translate, use ng-attr-label to handle the string inerpolation, example ng-attr-label="{{'CHECKBOX' | translate}}"
ng-model object none Controller property bound to the checkbox value.
ng-change (optional) function none Controller function to run when the checkbox value is updated. The value argument is exposed as val, example ng-change="myFunction(val)"
disabled (optional) boolean false Set the control as disabled
name (optional) string none Property name of the form under which the control is published.

Radio Buttons

The radio directive provides an easier implementation of the Design System styled radio button controls. The directive uses a element tag <pbds-radio></pbds-radio> with the following attributes.

Attribute Type Default Description
label string none The text label for the checkbox input. When using localized strings with angular-translate, use ng-attr-label to handle the string inerpolation, example ng-attr-label="{{'CHECKBOX' | translate}}"
ng-model object none Controller property bound to the checkbox value.
value string none The value to which the ngModel expression should be set when selected. Note that value only supports string values
ng-change (optional) function none Controller function to run when the checkbox value is updated. The value argument is exposed as val, example ng-change="myFunction(val)"
disabled (optional) boolean false Set the control as disabled
name (optional) string none Property name of the form under which the control is published.
