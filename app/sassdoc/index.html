    breakpoint

    @mixin breakpoint($point) { ... }

    Description

    Breakpoint mixins. The widths are customizable per project. These have been changed from defaults.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $point

    String representing sizes defined below

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Usage

    .media-thing {
  width:300px;
  @include breakpoint(screen-xs) {
    width: 100px;
  }
}

    CSS Output

    .media-thing {
  width: 300px;
}
  @media (max-width: 767px) {
    .media-thing {
      width: 100px;
    }
}

    Links

    • http://www.sitepoint.com/managing-responsive-breakpoints-sass/

    hide-text

    @mixin hide-text() { ... }

    Description

    Hides the text in an element, commonly used to show an image. Some elements will need block-level styles applied.

    Parameters

    None.

    Example

    Usage

    .element {
  @include hide-text;
}

    CSS Output

    .element {
  overflow: hidden;
  text-indent: 101%;
  white-space: nowrap;
}

    Links
